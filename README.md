# “简一智能翻译助手”隐私政策 / Privacy Policy for "简一智能翻译助手"

**最后更新日期 / Last Updated: [ 2026年4月21日]**

感谢您使用“简一智能翻译助手”（以下简称“本插件”）。我们深知隐私对您的重要性，并致力于保护您的数据安全。本隐私政策旨在向您说明本插件如何收集、使用和保护您的信息。
*Thank you for using "简一智能翻译助手" (hereinafter referred to as "the Extension"). We understand the importance of your privacy and are committed to protecting your data security. This Privacy Policy explains how the Extension collects, uses, and protects your information.*

## 1. 我们收集和使用哪些信息？(What information do we collect and use?)

本插件**不会**收集、追踪或传输您的任何个人身份信息（如姓名、联系方式、浏览历史记录等）。
*The Extension does NOT collect, track, or transmit any of your personally identifiable information (such as name, contact details, browsing history, etc.).*

为了实现核心翻译功能，本插件仅在以下特定场景处理您的数据：
*To provide its core translation functionalities, the Extension only processes your data in the following specific scenarios:*

*   **文本翻译 (Text Translation):** 当您使用“划词翻译”、“全页翻译”或“AI对话解释”功能时，本插件会获取您当前选中的文本或网页内容。这些文本将仅用于发送至第三方 AI 接口（DeepSeek API）以获取翻译和解释结果。
    *(When you use features like "Select to Translate," "Full Page Translation," or "AI Dialogue Explanation," the Extension accesses the text or web content you select. This text is solely sent to a third-party AI interface (DeepSeek API) to obtain translation and explanation results.)*
*   **本地存储 (Local Storage):** 当您使用“生词本”功能时，您的生词数据和插件配置信息仅会通过浏览器的 `storage` API 保存在您的本地设备上。**我们不会将这些本地数据上传至任何云端服务器，也不会与任何第三方共享。**
    *(When you use the "Vocabulary Book" feature, your vocabulary data and extension settings are saved locally on your device using the browser's `storage` API. **We do NOT upload this local data to any cloud servers, nor do we share it with any third parties.**)*

## 2. 第三方服务 (Third-Party Services)

本插件使用了 **DeepSeek API** 来提供翻译和 AI 对话服务。当您发起翻译请求时，相关文本将通过加密的 HTTPS 网络请求发送至 DeepSeek 服务器。请注意，第三方服务商的数据处理不受本隐私政策管辖，建议您查阅 DeepSeek 的官方隐私政策。
*The Extension uses the **DeepSeek API** to provide translation and AI dialogue services. When you initiate a translation request, the relevant text is sent to DeepSeek servers via secure HTTPS network requests. Please note that data processing by third-party service providers is not governed by this Privacy Policy, and we recommend reviewing DeepSeek's official privacy policy.*

## 3. 数据分享与披露 (Data Sharing and Disclosure)

我们**绝对不会**将您的任何数据出售、出租或交易给广告商或其他第三方。
*We **NEVER** sell, rent, or trade any of your data to advertisers or other third parties.*

## 4. 权限说明 (Permissions Explanation)

为了让插件正常工作，我们申请了以下浏览器权限：
*To function properly, we request the following browser permissions:*
*   `storage`: 用于在您的设备本地保存您的配置和生词本数据。(To save your settings and vocabulary book data locally on your device.)
*   `activeTab` / `scripting`: 用于在您当前浏览的网页上划词、读取文本并在页面上展示翻译结果。(To allow selecting text, reading content, and displaying translation results on your currently active web page.)
*   `host_permissions` (`https://api.deepseek.com/*`): 用于允许插件向 DeepSeek API 发送网络请求以获取翻译。(To allow the extension to send network requests to the DeepSeek API for translations.)
